微信小程序observe不生效

Posted by     "" on Monday, May 8, 2023

微信小程序中,自定义的组件components的页面,写一个observe监听器,但是不生效,之前也遇到过类似的问题,用一些小程序组件特有的功能(比如组件的生命周期)也不生效 研究后发现是因为组件的js页面最外层那个大括号一般的页面是写Page,但组件中的话要改成Component

​    Component({
  properties: {
  	 productList: {
      type: Array,
      value: [],
    },
  },
  data: {
	dataVal:''
  },
  lifetimes: {
  	 attached() {
      const {dataVal,productList} = this.data
		this.setData({
			dataVal:"zhou",
			productList:[]
		})
    },
  },
  refresh(){
    //想要读取 properties的值应该这么写
    console.log(this.properties.productList)
    }
  observers: {
    productList: function (val) {//监听productList变动
      console.log(val)
    },
    '**': function() {// 每次 setData 都触发
      console.log("this.data")
    },
  },
  
});
