高德显示xxx is not a construct

Posted by     "huoguo32" on Wednesday, March 8, 2023

高德在vue中常见的的组件加载问题,xxx组件is not a constructor

在创建高德地图时引入插件.有时候会没用.网上说的引入sdk的方法也不适用于vue 解决方案通过AMap.plugin单独异步加载,加载后再执行new AMap.xxx():

//不需要声明成响应式数据
 let geocoder

 AMap.plugin('AMap.Geocoder', function () {
        geocoder = new AMap.Geocoder()
      })

 geocoder.getLocation(searchText.value, (status, result) => {
        if (status === 'complete' && result.geocodes.length) {
          const position = result.geocodes[0].location
          props.mapInstance.setCenter(position)
          const marker = new props.mapInstance.Marker({
            position
          })
          props.mapInstance.add(marker)
        }
      })
